# Juego del Amigo Secreto
Aplicacion simple para jugar al amigo secreto.

## Funcionalidades
- Agregar uno a uno a los participantes.
- Asgnacion aleatoria del amigo secreto.
- Mostrar el resultado.
- Opcion para reiniciar el juego.

## Lenguajes
- HTML
- CSS
- JavaScript
